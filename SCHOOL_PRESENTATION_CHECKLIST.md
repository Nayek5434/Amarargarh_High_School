# School Presentation Checklist (Principal + Teachers + Students)

Use this checklist to present your website professionally.

## 1) Before presentation day

- Prepare final URL (IP/domain)
- Confirm internet is stable
- Keep one backup hotspot
- Login to admin once before demo
- Keep sample content ready:
  - 2 notices
  - 2 events
  - 2 magazine posts
  - 1 teacher update

## 2) Demo script (10–12 minutes)

### Minute 1–2: Opening

- Explain objective: one official digital platform for school communication.
- Mention features: notices, events, admissions, teachers, achievements, magazine.

### Minute 3–5: Public website flow

- Home page overview
- Theme toggle demonstration
- Mobile menu demonstration
- Open Notices, Events, Teachers, Achievements

### Minute 6–8: Admin dashboard flow

- Open `/admin`
- Show secure login
- Update one notice
- Add one event
- Refresh public page and show instant update

### Minute 9–10: Security and reliability

- Mention secure admin sessions
- Mention login attempt controls/recovery flow
- Mention regular backup and update process

### Minute 11–12: Closure

- Explain expected impact:
  - faster parent communication
  - centralized updates
  - better public image for school

## 3) Slide structure (recommended)

1. Problem today (manual notice gap)
2. Solution (official website)
3. Features snapshot
4. Live demo screenshots
5. Security and reliability
6. Cost (free-tier hosting)
7. Rollout plan
8. Q&A

## 4) Rollout plan to tell school

- Week 1: content onboarding and admin training
- Week 2: pilot with limited notice/event updates
- Week 3: full public launch
- Week 4+: monthly review and improvements

## 5) Roles to propose

- One primary admin (school office)
- One backup admin (teacher)
- One reviewer (principal office)

## 6) What to avoid in presentation

- Do not show secrets (`.env`, passwords)
- Do not edit database files directly in demo
- Do not perform risky server commands live

## 7) Day-of quick test list

- Home page loads
- Theme toggle works
- Mobile menu works
- Admin login works
- Security/Pages/Events/Notices section links work
- Add and delete one sample event successfully
- Logout works

## 8) Success message for school committee

"This website gives Amarargarh High School a reliable official digital channel for notices, events, and academic communication, with secure admin control and near-zero hosting cost under student/free-tier usage."
